# GeeksHub - Co-working Space Website

GeeksHub is a dynamic website created for a co-working space, designed to enhance user experience while promoting sustainability by achieving a 100% reduction in paper waste through digital records.

## Features

- **User-Friendly Interface:** Intuitive design for seamless navigation and interaction.
- **Digital Record Management:** Eliminates paper logs, supporting eco-friendly practices.
- **Team Collaboration:** Efficient task distribution and organization using Trello.

## Technologies Used

- **Frontend:** ReactJS, TailwindCSS
- **Backend:** Firebase for data management
- **Design Patterns:** Active Record design pattern, BaaS architectural pattern

## Getting Started

To get started with GeeksHub, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/geekshub.git
    ```
2. Navigate to the project directory:
    ```bash
    cd geekshub
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
4. Run the app:
    ```bash
    npm start
    ```
